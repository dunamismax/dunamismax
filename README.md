## Principles

> "Boring infrastructure is the highest compliment. Novel architecture is a liability until proven otherwise."

- **Ship small, verify always.** Lint, typecheck, build, test — every change, no exceptions
- **Self-hostable by default.** Can't run it yourself? You don't own it
- **Right tool for the job.** TypeScript is the default. Python, Rust, Go when they genuinely win
- **Mirror everything.** All repos on [GitHub](https://github.com/dunamismax) + [Codeberg](https://codeberg.org/dunamismax)
- **Complexity must be earned.** Every abstraction needs justification beyond "might be useful someday"
