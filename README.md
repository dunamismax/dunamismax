## Quick Start

### Prerequisites

- Git
- A GitHub account for viewing profile-repo rendering behavior

### Clone and Review

```bash
git clone https://github.com/dunamismax/dunamismax.git
cd dunamismax
```

Expected result: this README is the primary content for the GitHub profile page at `https://github.com/dunamismax`.

## Features

- Central snapshot of active projects and engineering focus areas.
- Concise public profile entry point for collaborators and recruiters.
- Lightweight repository with no build pipeline or runtime dependencies.
